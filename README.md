# AccountBook-V2
简单的记账系统：使用Maven作为项目管理，SSM（Spring，SpringMVC，MyBatis）开发框架。

功能主要包括：


##用户
 - 注册，验证码
 - 登录
 - 记住密码，自动登录
 
## 收入
 - 添加收入 
 - 显示收入的简单统计 
 - 细目（修改，删除） 
 
## 支出
 - 添加支出 
 - 显示支出的简单统计
 - 细目（修改，删除）
 
##往月收支
 - 查看往月收支细目
 - 按月查看
 - 搜索
 - 导出到excel (?)
 
##项目管理
 - 添加项目 
 - 删除 
 - 修改
